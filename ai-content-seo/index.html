Overview

AI Content + SEO Assistant (AICSA) is a professional WordPress plugin designed to help website owners, marketers, and developers generate high-quality, SEO-optimized content directly inside the WordPress editor and popular page builders.

The plugin integrates with multiple AI providers and focuses on clean output, search-engine friendly structure, and editorial control.

System Requirements

  • WordPress 5.8 or higher
  • PHP 7.4 or higher (PHP 8.x supported)
  • HTTPS enabled (required for API calls)
  • One or more supported AI providers (optional)

Supported Editors & Builders

  • Gutenberg (Block Editor)
  • Classic Editor
  • Elementor
  • Beaver Builder
  • WPBakery (Visual Composer)

Installation

Step 1: Upload Plugin

  • Go to WordPress Admin → Plugins → Add New
  • Click Upload Plugin
  • Select the plugin ZIP file
  • Click Install Now and then Activate

Step 2: Verify Installation

After activation, you will see a new menu item: AI Content SEO Assistant in the WordPress admin sidebar.

Initial Configuration

Global Settings

Navigate to: AI Content SEO Assistant → Settings

  • Select your active AI provider
  • Set maximum token / word limits
  • Choose enabled post types

API Keys

The plugin does not include built-in API keys. You must provide your own keys for supported providers.

  • OpenAI API Key
  • Google Gemini API Key
  • Groq API Key

API keys are stored securely using WordPress options and are never shared externally.

Using the Plugin

Classic Editor

When editing a post or page, you will see the AI Content SEO Assistant metabox in the sidebar.

  • Enter your content prompt
  • Select content action (Write, Rewrite, Improve, etc.)
  • Click Generate
  • Content is inserted directly into the editor

Gutenberg (Block Editor)

In Gutenberg, the plugin integrates via a sidebar panel.

  • Select a text block
  • Open the AI assistant panel
  • Generate or rewrite content

Elementor

The plugin provides a dedicated Elementor widget.

  • Add the AI Content Writer widget
  • Enter your prompt
  • Generate content inside Elementor

Beaver Builder & WPBakery

For visual builders, the plugin detects active editor mode and injects content safely without breaking layouts.

SEO-Focused Content Generation

How AICSA Helps SEO

  • Semantic keyword structure
  • Proper heading hierarchy (H1–H6)
  • Readable paragraph length
  • Natural keyword placement
  • Human-like tone (not spammy)

Best Practices for SEO

  • Always include a clear topic in your prompt
  • Specify target keywords explicitly
  • Use “SEO-friendly” or “search optimized” in prompts
  • Generate section-by-section for long articles

Example SEO Prompt

Write a 1200-word SEO-optimized article about "On-Page SEO for WordPress".
Use H2 and H3 headings, short paragraphs, and a professional tone.

Word Limit & Formatting Control

AICSA allows precise control over output length.

  • Maximum word limits enforced
  • Automatic trimming if provider exceeds limit
  • Formatting preserved (lists, headings, paragraphs)
