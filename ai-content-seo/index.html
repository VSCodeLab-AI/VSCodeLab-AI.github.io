WordPress Plugin

AI Content + SEO Assistant

A multi-provider AI content generator for WordPress supporting OpenAI, Gemini, and Groq.

Key Features

  • AI-powered content generation inside WordPress editors
  • Supports OpenAI, Google Gemini, and Groq APIs
  • Compatible with Gutenberg, Classic Editor, Elementor, Beaver Builder, and WPBakery
  • Secure API handling and sanitized inputs
  • Optimized for performance and Envato standards

External Services

  • OpenAI API (user-provided key)
  • Google Gemini API (user-provided key)
  • Groq API (user-provided key)
  • Chart.js (loaded via CDN for analytics visualization)
